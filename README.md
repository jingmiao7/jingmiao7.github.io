# Jing Miao Personal Website

This repository contains the source for Jing Miao's GitHub Pages website.

## Main Content

- Homepage text: `_pages/about.md`
- Site identity, profile photo, and sidebar links: `_config.yml`
- Top navigation: `_data/navigation.yml`
- CV and other documents: `files/`
- Images: `images/`

## Local Preview

Install the Ruby dependencies, then run:

```bash
bundle install
bundle exec jekyll serve
```

The site is published from the `master` branch by GitHub Pages.
